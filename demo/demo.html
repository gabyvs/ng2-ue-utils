Demo App for Angular2 Unified Experience Utilities

Loading Dots

A directive which animates an ellipsis to indicate that a section of your page is still being loaded

Date Moment Pipe

A pipe which converts timestamps into formatted dates - angular2 native date pipe was broken for some browsers on RC2
{{ today | dateMoment }}

From Now Pipe

A pipe which converts timestamps into relative time strings
{{ recent | fromNow }}
